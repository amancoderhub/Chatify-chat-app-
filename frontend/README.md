# 💬 chatify - Real-Time Chat Application

A full-stack real-time chat application built using **MERN Stack, TypeScript, and Socket.io**.

---

## 🚀 Features

* 🔐 User Authentication (JWT)
* 💬 Real-time messaging using Socket.io
* 🟢 Online/Offline user status
* 📨 One-to-one chat system
* ⚡ Fast UI with Vite + React + TypeScript
* 🌐 REST API with Express & MongoDB

---

## 🛠️ Tech Stack

### Frontend

* React (TypeScript)
* Vite
* Tailwind CSS (optional)

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* Socket.io
